# Changelog

## [1.0.0] - 2026-06-17

### Added
- Initial public release
- Excel workbook with 67 sheets including Master_Index (2,660 records), Dashboard, Link_Audit, Recommended_2026, Free_GIS_Data, and 30+ category sheets
- Flask web app with search, filter, and status badges
- Static GitHub Pages demo (docs/index.html)
- Python export script for CSV/JSON
- Python URL link checker with polite request behavior
- Free GIS Data updater script
- Release packaging script with SHA256 checksums
- GeoJSON map of documented GIS data sources
- Bilingual documentation (English / العربية)
- MIT License for code; CC BY 4.0 recommended for curated index

### Notes
- GeoJSON coordinates are symbolic/placeholder and should not be used for spatial analysis
- Link audit results reflect the date of last check; URLs may change
